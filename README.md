### What I Ship

| Project | Stack | What It Does |
|:--------|:------|:-------------|
| **MAIRS** | LangGraph, Neo4j, Pinecone, BGE Reranker, DistilBERT, FastAPI | Multi-agent investment research system with sentiment analysis and graph-augmented retrieval |
